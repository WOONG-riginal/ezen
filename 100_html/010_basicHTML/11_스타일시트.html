CSS(CascadingStyleSheet)

웹표준에 의한 웹문서는 디자인과 내용이 분리되어 있습니다.

내용은 HTML을 이용해 구성하고, 디자인은 CSS를 이용해 꾸미는 것입니다.

인라인 스타일시트 적용하기

class 스타일 시트

class로 스타일 정의시 구분기호로 .을 사용

스타일 캐스캐이딩 원리

같은 이름의 선택자에 같은 속성명이 정의되어 있을 때,
속성값이 다르다면 하단에 있는 스타일의 속성값이 적용됨

ID 스타일시트

id로 스타일 정의시 구분기호로 #을 사용

그룹 선택자

그룹 선택자 구분기호는 ,를 사용하여 여러개의 요소를 동시에 선택
